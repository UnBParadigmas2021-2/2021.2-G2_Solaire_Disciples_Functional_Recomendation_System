## Instalação 
**Linguagens**: Haskell<br>
**Tecnologias**: Cabal, Hamlet e Scotty<br>

Instação no Ubuntu
```shell
$ sudo apt-get install -y cabal-install
$ cabal update
$ cabal install
```

## Uso 
```shell
$ cabal run
```
